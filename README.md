# Apple Stock Analysis 2020-2024

Analysis of Apple (AAPL) stock performance from 2020 to 2024, built with Python, Pandas, and Matplotlib.

## What this project covers
- Historical closing price with key market events annotated (COVID Crash, Fed Rate Hike)
- 30-day rolling volatility to measure market risk over time
- Portfolio simulation: 1000 CHF invested on January 1st 2020

## Key findings
- Average daily return: 0.12%
- Average daily volatility: 1.95%
- 1000 CHF invested in January 2020 grew to ~2650 CHF by end of 2023 (+165%)
- COVID Crash (March 2020) produced a volatility peak 3x above average

## Tools
- Python, Pandas, Matplotlib, yfinance
